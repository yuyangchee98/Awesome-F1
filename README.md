## Data & APIs
- [Jolpica F1](https://github.com/jolpica/jolpica-f1) - Open source F1 API, Ergast replacement
- [OpenF1](https://github.com/br-g/openf1) - Real-time and historical F1 data
- [F1DB](https://github.com/f1db/f1db) - Comprehensive F1 database

## Libraries & SDKs
- [FastF1](https://github.com/theOehrly/Fast-F1) - Python package for F1 data
- [f1dataR](https://github.com/SCasanova/f1dataR) - R package for F1 data

## Fan Tools
- [F1 Points Calculator](https://github.com/yuyangchee98/F1-Points-Calculator) - An [interactive championship points calculator](https://f1pointscalculator.chyuang.com/) and race simulator for predicting F1 standings.
