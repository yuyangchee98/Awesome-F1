## Data & APIs
- [Jolpica F1](https://github.com/jolpica/jolpica-f1) - Open source F1 API, Ergast replacement
- [OpenF1](https://github.com/br-g/openf1) - Real-time and historical F1 data
- [F1DB](https://github.com/f1db/f1db) - Comprehensive F1 database

- [Formula1 Datasets](https://github.com/toUpperCase78/formula1-datasets) - Datasets and analyses for F1 World Championship
- [F1 Circuits](https://github.com/bacinger/f1-circuits) - Formula 1 circuit tracks data in GeoJSON format
## Libraries & SDKs
- [FastF1](https://github.com/theOehrly/Fast-F1) - Python package for F1 data
- [f1dataR](https://github.com/SCasanova/f1dataR) - R package for F1 data
- [ergast-py](https://github.com/Samuel-Roach/ergast-py) - Python wrapper for the Ergast API
## Fan Tools
- [F1 Points Calculator](https://github.com/yuyangchee98/F1-Points-Calculator) - An [interactive championship points calculator](https://f1pointscalculator.chyuang.com/) and race simulator for predicting F1 standings.
- [RaceControl](https://github.com/robvdpol/RaceControl) - Standalone F1TV client for Windows
- [f1viewer](https://github.com/SoMuchForSubtlety/f1viewer) - TUI (Terminal User Interface) for F1TV
- [F1 Calendar](https://github.com/sportstimes/f1) - Formula 1 race calendar with email reminders and push notifications
- [F1 Web Viewer](https://github.com/bootsie123/F1-Web-Viewer) - Grid-based web viewer for F1TV with custom layouts
## Visualization
- [F1 Race Replay](https://github.com/IAmTomShaw/f1-race-replay) - An interactive race visualisation and data analysis tool built with Python
- [F1 Dash](https://github.com/slowlydev/f1-dash) - A real-time F1 dashboard
- [Formula 1 Lap Charts](https://github.com/davidor/formula1-lap-charts) - Interactive lap chart visualization of F1 races
- [Fastlytics](https://github.com/subhashhhhhh/Fastlytics) - Modern web app for F1 data analytics with race results, standings, and telemetry charts
## Mobile Apps
- [F1AppleTV](https://github.com/NoahFetz/F1AppleTV) - F1TV client for Apple TV
- [Undercut F1](https://github.com/JustAman62/undercut-f1) - F1 Live Timing TUI with variable delay to sync to TV
