# Contributing to Awesome-F1

Thanks for your interest in contributing!

## Adding a Project

Fork this repo, add your project to README.md using this format:

```
- [Project Name](https://github.com/user/repo) - Brief description
```

Then submit a pull request.

## Requirements

Projects should be related to Formula 1, open source, and reasonably maintained.

## Questions?

Open an issue for any questions or suggestions.
